# DunDun GitHub Pages

Bilingual GitHub Pages for DunDun:

- `support/index.html`
- `privacy/index.html`

## Before publishing
Replace every instance of `YOUR_SUPPORT_EMAIL@example.com` with your official support email.

This draft assumes:
- data is stored locally by default;
- optional iCloud sync uses the user's personal iCloud account;
- camera permission is requested only for taking photos;
- photo permission is requested only for uploading photos;
- selected-photo access is supported;
- no ads, analytics SDKs, crash-reporting SDKs, user accounts, or developer-operated servers are used.

## Typical GitHub Pages URLs
- `https://YOUR_USERNAME.github.io/YOUR_REPO/support/`
- `https://YOUR_USERNAME.github.io/YOUR_REPO/privacy/`
